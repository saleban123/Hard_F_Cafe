Frost Mage

Caster • CC Master • Burst Specialist

Master of crowd control with Poly chains, Deep + Orb combos, and perfect positioning. The ultimate CC playmaker for RBG teams.

Progression Guide

Master Frost Mage from 1300 rating to R1 with these key improvements at each bracket

1300-1400

Foundation

Key Focus: Polymorph enemy healers on cooldown

  • Learn basic Poly timing and positioning
  • Understand DR tracking basics
  • Practice max range positioning
1400-1500

Burst Combos

Key Focus: Deep + Orb combos for burst

  • Master Deep Freeze + Frozen Orb timing
  • Learn to save burst for kill windows
  • Practice Ice Block usage
1500-1600

CC Chains

Key Focus: Ring choke points, CC chains

  • Use Ring of Frost to control positioning
  • Chain CC with teammates
  • Learn when to use Blink defensively
1600-1700

Team Coordination

Key Focus: Sync Polys with team stuns

  • Coordinate CC with DK Grip + stuns
  • Call out Poly targets
  • Learn to read teamfight flow
1700-1800

Burst Timing

Key Focus: Save Orb for Deep goes

  • Never waste Frozen Orb randomly
  • Time burst with team cooldowns
  • Learn to kite melee pressure
1800-1900

Positioning

Key Focus: Play max range, avoid swaps

  • Master max range positioning
  • Learn to read incoming swaps
  • Use Blink to escape pressure
1900-2000

Defensive Usage

Key Focus: Save Block for kill swaps

  • Ice Block only when necessary
  • Learn to pre-position for swaps
  • Master trinket usage
2000-2100

Trinket Discipline

Key Focus: Trinket wisely, not random

  • Only trinket when it saves your life
  • Learn to read enemy burst windows
  • Coordinate with healers
2100-2200

Balance Mastery

Key Focus: Balance damage + CC uptime

  • Perfect balance of damage and CC
  • Learn to read teamfight priorities
  • Master all utility spells
2200-2300

Perfect Chains

Key Focus: Perfect Ring + Poly chains

  • Flawless CC chain execution
  • Perfect positioning in all situations
  • Master all defensive cooldowns
2300-2400

Leadership

Key Focus: Call CC rotations

  • Lead team CC rotations
  • Call out targets and timing
  • Coordinate with all teammates
2400-2500

Offensive Leadership

Key Focus: Lead CC chains offensively

  • Initiate offensive CC chains
  • Create kill opportunities
  • Master all advanced techniques
2500-2600 R1

R1 Mastery

Key Focus: Perfect CC/burst balance, leads mage playmaking

  • Flawless execution in all situations
  • Leads entire team's CC strategy
  • Creates game-winning plays
  • Perfect balance of offense and defense

R1 Behaviors & Coaching Points

Positioning

  • Anchor positioning: Max range, never frontline
  • Pre-positioning: Always ready for swaps
  • Blink mastery: Perfect escape timing
  • Ring placement: Choke points or defensive peel

CC Management

  • Poly bot: Never lets healer breathe
  • DR tracking: Perfect timing on all CC
  • Chain coordination: Works with all teammates
  • Call rotations: Leads team CC strategy

Burst Execution

  • Deep + Orb combos: Only in CC windows
  • Timing mastery: Perfect burst coordination
  • Resource management: Never waste cooldowns
  • Target selection: Always optimal choices

Defensive Play

  • Ice Block discipline: Only when necessary
  • Trinket usage: Perfect timing
  • Peel role: Novas/CoC to help teammates
  • Kiting mastery: Never dies to melee

Key Abilities & Usage

Polymorph

Your primary CC tool. Use on healers constantly, chain with team stuns, and never let enemies breathe.

R1 Tips: Call out targets, track DRs, use defensively to peel

Deep Freeze + Frozen Orb

Your burst combo. Only use when you can guarantee the kill or force major cooldowns.

R1 Tips: Save for team goes, coordinate with stuns, never waste

Ring of Frost

Area control tool. Use to choke points, peel for teammates, or set up teamfights.

R1 Tips: Perfect placement, defensive usage, team coordination

Ice Block

Your "oh shit" button. Use only when absolutely necessary, not as a crutch.

R1 Tips: Pre-positioning > Ice Block, save for swaps, coordinate with healers
