Holy Priest

Healer • Mass Dispel Master • Clutch Savior

Master of clutch saves with Mass Dispel, Guardian Spirit, and Grip mastery. The ultimate defensive playmaker for RBG teams.

Progression Guide

Master Holy Priest from 1300 rating to R1 with these key improvements at each bracket

1300-1400

Foundation

Key Focus: Learn basic healing rotation and positioning

  • Master basic healing spells (Heal, Flash Heal, Prayer of Healing)
  • Understand midline positioning
  • Learn to use Mass Dispel on bubbles/blocks
1400-1500

Guardian Spirit

Key Focus: Use Guardian Spirit proactively

  • Learn to pre-GS before lethal damage
  • Master clutch save timing
  • Coordinate with other healers
1500-1600

Grip Mastery

Key Focus: Use Grip to save allies or set up kills

  • Grip allies out of danger zones
  • Grip enemies into CC chains
  • Learn defensive vs offensive Grip usage
1600-1700

Mass Dispel Timing

Key Focus: Perfect Mass Dispel usage

  • Always ready for Bubble/Block
  • Learn to read enemy cooldowns
  • Coordinate with team for clutch saves
1700-1800

CD Management

Key Focus: Stagger defensive cooldowns

  • Learn to stagger GS, Desperate Prayer, and Power Word: Shield
  • Master defensive rotation
  • Understand CD priority in different situations
1800-1900

Spectral Guise

Key Focus: Use Spectral Guise to break CC or set up Fear

  • Break CC chains with Spectral Guise
  • Set up Fear on enemies
  • Learn offensive vs defensive usage
1900-2000

Positioning Mastery

Key Focus: Perfect midline positioning

  • Master max range positioning
  • Learn to read incoming swaps
  • Use Grip to escape pressure
2000-2100

Team Coordination

Key Focus: Coordinate with all teammates

  • Call out Mass Dispel targets
  • Coordinate GS with other healers
  • Lead defensive rotations
2100-2200

Perfect Timing

Key Focus: Perfect timing on all abilities

  • Perfect Mass Dispel timing
  • Perfect GS usage
  • Perfect Grip timing
2200-2300

Offensive Utility

Key Focus: Use utility offensively

  • Grip enemies into kill setups
  • Use Spectral Guise for offensive positioning
  • Coordinate with team for offensive plays
2300-2400

Leadership

Key Focus: Lead team's defensive strategy

  • Call out all defensive rotations
  • Lead Mass Dispel coordination
  • Master strategic thinking
2400-2500

Playmaking

Key Focus: Create game-winning plays

  • Create clutch saves with perfect timing
  • Set up kills with Grip
  • Master all advanced techniques
2500-2600 R1

R1 Mastery

Key Focus: Mass Dispel + Guardian Spirit mastery wins games

  • Flawless execution in all situations
  • Leads entire team's defensive strategy
  • Creates game-winning plays
  • Perfect balance of healing and utility

R1 Behaviors & Coaching Points

Positioning

  • Midline positioning: Between team and backline
  • Pre-positioning: Always ready for swaps
  • Grip positioning: Perfect angles for saves
  • Mass Dispel range: Always in range for clutch saves

Mass Dispel Mastery

  • Mass Dispel timing: Always ready for Bubble/Block
  • Clutch saves: Perfect timing on all dispels
  • Team coordination: Call out dispel targets
  • Strategic thinking: Read enemy cooldowns

Guardian Spirit

  • Preemptive GS: Guardian Spirit before lethal damage
  • Clutch timing: Perfect save execution
  • Team coordination: Coordinate with other healers
  • Resource management: Never waste GS

Grip & Spectral Guise

  • Grip mastery: Save allies from swaps or Grip enemies into CC chains
  • Spectral Guise: Breaks CC chains or sets up Fear
  • Offensive utility: Use abilities for kills
  • Defensive utility: Use abilities for saves

Key Abilities & Usage

Mass Dispel

Your primary utility tool. Use to clutch save teammates from bubbles, blocks, and other immunities.

R1 Tips: Always ready for Bubble/Block, coordinate with team, clutch saves

Guardian Spirit

Your clutch save tool. Use before lethal damage to save teammates from certain death.

R1 Tips: Preemptive usage, perfect timing, coordinate with healers

Grip

Your positioning tool. Use to save allies from swaps or Grip enemies into CC chains.

R1 Tips: Save allies from swaps, Grip enemies into CC, perfect timing

Spectral Guise

Your escape tool. Use to break CC chains or set up Fear on enemies.

R1 Tips: Break CC chains, set up Fear, perfect timing
